# HTTP Status Codes Cheat Sheet

HTTP status codes indicate the result of an HTTP request.

## 1xx Informational

100 Continue  
101 Switching Protocols

---

## 2xx Success

200 OK  
201 Created  
204 No Content

---

## 3xx Redirection

301 Moved Permanently  
302 Found  
304 Not Modified

---

## 4xx Client Errors

400 Bad Request  
401 Unauthorized  
403 Forbidden  
404 Not Found

---

## 5xx Server Errors

500 Internal Server Error  
502 Bad Gateway  
503 Service Unavailable
